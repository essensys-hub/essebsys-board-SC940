# SC940 Board Project

Welcome to the **Essensys SC940 Board** project repository. This project contains the hardware design and firmware source code for the SC940 board.

## Documentation

Full documentation for this project is available and can be built using MkDocs.
It covers:
- **Hardware**: Schematics, PCB layouts, and manufacturing files.
- **Firmware**: Source code and programming guides.

### Viewing Documentation Locally
To view the documentation locally:

1. Install MkDocs and the Material theme:
   ```bash
   pip install mkdocs-material
   ```
2. Run the documentation server:
   ```bash
   mkdocs serve
   ```
3. Open your browser to `http://127.0.0.1:8000`.

## Project Structure

- `SC940D/`: Electronic design files (Altium Designer), including Schematics and PCB.
- `SC940D/Prog/`: Firmware source code and hex files.
- `docs/`: Documentation source files.
- `mkdocs.yml`: MkDocs configuration.
