# Welcome to SC940 Board Documentation

This documentation covers the hardware design and firmware implementation for the SC940 board project.

## Repository Structure

The project is organized as follows:

- **SC940D/**: Main project directory containing hardware design files.
    - **Schematics**: Electronic schematics (`.SchDoc`).
    - **PCB Layout**: PCB design files (`.PcbDoc`).
    - **Gerbers**: Manufacturing files.
    - **Prog**: Firmware source code and programming resources.
