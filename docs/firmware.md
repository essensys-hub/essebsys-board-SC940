## Programing Guide

Refer to `Fiche programmation SC940 v1.docx` for detailed programming instructions.

## Source Code

The source code for the firmware is in the `code_ba (fichiers sources 1.7)` directory.
Pre-compiled hex files are also available, such as `ba v1_7 piece vie.hex`.
